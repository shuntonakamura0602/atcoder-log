## ABC088 B  
$$
N枚のカードがあります。i枚目のカードにはa_iという整数が書かれています。
$$
$$
AliceとBobは交互にカードを一枚ずつ取っていきます。Aliceが先手です。
$$
$$
2人が最適な戦略をとった時AliceがBobより何点多く取るか求めてください
$$

```python
N = int(input())
a = list(map(int,input().split()))

a.sort(reverse=True)

result = 0

for i in range(N):
    if i%2 == 0:
        result += a[i]
    else:
        result -= a[i]

print(result)
```
